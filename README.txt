CHB Affiliate Community PWA (v3 - no-module)
=====================================

This build works even if you double-click index.html (file://) because it does NOT use ES modules.

1) Open index.html
2) Login in Demo Mode with ANY email + ANY PIN
3) To connect Google Sheets:
   - Open app-bundle.js
   - Set CONFIG.API_BASE to your Apps Script Web App URL

Note: If your Apps Script is live, login requires matching Users sheet (email, pin, status=active).
